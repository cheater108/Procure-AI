# Procure AI Frontend
This is the frontend for Procure AI, an AI assisted procurement tool which helps in creating Request for Proposals (RFPs), send emails and rank proposals.

## Project Setup

### Prequisites
- Node version atleast v20.

### Install steps

Install dependencies

```bash
  npm install
```

Create .env file see .env.example

Run the dev server

```bash
  npm run dev
```
